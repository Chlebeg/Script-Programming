Przed korzystaniem nalezy zainstalowac "cryptography.lib" komenda
    pip install cryptography

By korzystac z czatu nalezy wpisać podane komendy

- Dla serwera:
python .\server.py
Serwer po odpaleniu zwraca informacje o kluczu w postaci
    "b'<key>'"
potrzebna nam sama wartość key

- Dla clientów:
    python .\client.py
Client po odpaleniu prosi o informacje o kluczu i o nazwie uzytkownika
